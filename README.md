# PET中心智能助手 Skill

## 简介

为医院PET中心打造的AI智能咨询助手，参考"金谷园饺子馆Skill"架构设计，专注于为患者和临床医生提供PET检查相关的专业信息服务。

## 功能覆盖

### 支持的显像剂（9种）

| 显像剂 | 类型 | 主要应用 |
|--------|------|---------|
| FDG | 代谢显像 | 肿瘤、心肌存活、脑代谢 |
| AV45 | 淀粉样蛋白 | 阿尔茨海默病 |
| AV1451 | Tau蛋白 | 阿尔茨海默病分期 |
| FCIT | 多巴胺能 | 帕金森病 |
| PSMA | 前列腺特异性 | 前列腺癌 |
| 奥曲肽 | 生长抑素受体 | 神经内分泌瘤 |
| FAPI | 成纤维细胞活化蛋白 | 肿瘤/心肌纤维化 |
| FES | 雌激素受体 | 乳腺癌ER评估 |
| DOPA | 氨基酸代谢 | 脑胶质瘤、NET |

### 核心功能

1. **检查前准备指导** — 按显像剂类型提供详细准备要求
2. **流程说明** — 从预约到取报告的完整流程
3. **显像剂信息** — 适应症、剂量、等待时间
4. **常见问题解答** — 辐射安全、禁忌症、注意事项
5. **剂量计算** — 根据体重计算推荐剂量
6. **科室信息查询** — 位置、设备、联系方式

## 使用方式

### 作为OpenClaw Skill使用

1. 将本目录打包为skill包
2. 通过OpenClaw加载使用
3. 用户可通过自然语言触发咨询

### 直接查询

用户可通过以下方式获取信息：
- "我要做FDG检查，需要准备什么？"
- "FAPI和FDG有什么区别？"
- "检查要多长时间？"
- "孕妇能做PET吗？"

## 设计特点

### 参考金谷园Skill架构
- **SKILL.md** — 核心指令文件，包含触发场景、语气指南、盲区应对
- **skill.json** — 机器可读配置，定义工具和系统提示词
- **references/** — 深度参考文档，供Agent查询使用

### 语气与调性
- **专业、清晰、安心** — 像经验丰富的核医学技师
- 说人话，少用术语，用到时解释
- 给出明确指令（"禁食6小时"而非"适当禁食"）
- 不确定时说"建议咨询您的主治医生"

### 安全原则
- 绝不编造药物剂量和扫描参数
- 涉及禁忌症时严谨明确
- 不替代医生诊疗建议
