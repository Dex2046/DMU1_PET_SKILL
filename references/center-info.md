## 科室布局

```
入口 → 大厅（服务台）→ 诊室
                    ↓
              注射室 → 静卧室
                    ↓
        PET/CT扫描室 / PET/MR扫描室 → 出口
```

## 检查流程

1. **现场预约** → 联合路院区核医学诊疗中心预约
2. **到科报到** → 服务台登记，核对信息
3. **医生问诊** → 了解病史、用药情况，签署知情同意书
4. **注射显像剂** → 按体重/固定剂量静脉注射
5. **静卧等待** → 在静卧室安静休息（时间因显像剂而异，通常约1小时）
6. **扫描检查** → 平躺于检查床，配合技师完成扫描
7. **检查结束** → 离开，按注意事项执行
8. **获取报告** → 文字报告手机查阅，电子胶片扫描二维码
