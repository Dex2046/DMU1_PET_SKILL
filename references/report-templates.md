# PET报告模板与术语规范

## 一、报告基本结构

### 标准报告格式

```
【检查信息】
- 患者姓名、性别、年龄、ID号
- 检查日期
- 显像剂种类及剂量
- 注射时间
- 显像时间

【临床诊断】
（申请单提供的临床诊断）

【检查所见】
（描述性文字，按部位系统描述）

【诊断意见】
（结论性文字，分点列出）

【建议】
（随访、进一步检查等）

【报告医师】
【审核医师】
【报告日期】
```

## 二、常用术语规范

### SUV（标准化摄取值）
- **SUVmax：** 最大标准化摄取值，最常用半定量指标
- **SUVmean：** 平均标准化摄取值
- **SUVpeak：** 峰值标准化摄取值（1cm³体积内平均值）

### 常用判读标准

#### Deauville评分（淋巴瘤疗效评估）
| 评分 | 定义 |
|------|------|
| 1分 | 无摄取 |
| 2分 | 摄取 ≤ 纵隔血池 |
| 3分 | 摄取 > 纵隔血池但 ≤ 肝血池 |
| 4分 | 摄取 > 肝血池 |
| 5分 | 摄取 >> 肝血池（明显增高） |
| X分 | 新的摄取灶 |

- 1-3分：考虑完全代谢缓解
- 4-5分：考虑残留病灶

#### PERCIST标准（实体瘤疗效评估）
- **完全代谢缓解(CMR)：** 所有病灶消失，SUV降至正常背景水平
- **部分代谢缓解(PMR)：** SUV下降≥30%
- **代谢稳定(SMD)：** 不符合CMR/PMR/PMU
- **代谢进展(PMU)：** SUV升高≥30%或出现新病灶

#### Lugano分类（淋巴瘤分期）
- 基于PET/CT的分期系统
- 结合解剖分期和代谢反应评估

## 三、各项目报告要点

### FDG肿瘤显像

**检查所见要点：**
1. 描述生理性摄取（脑、心肌、肾、膀胱、声带等）
2. 按部位描述异常摄取灶：
   - 位置（解剖定位）
   - 大小（CT测量）
   - SUVmax值
   - 形态特征
3. 与既往检查对比（如有）

**诊断意见模板：**
```
1. [部位]见[数目]个FDG代谢异常增高灶，SUVmax约[数值]，考虑为[诊断]。
2. [其他发现]。
3. 与[日期]PET/CT比较，[变化描述]。
```

### 心肌灌注/代谢显像

**检查所见要点：**
1. 左心室心肌FDG摄取分布
2. 灌注-代谢匹配/不匹配分析
3. 心腔大小、室壁厚度

**诊断意见模板：**
```
1. 左心室心肌FDG摄取[均匀/不均匀]，[部位]见代谢减低区。
2. 结合灌注显像，[部位]呈[灌注-代谢匹配/不匹配]，提示[存活心肌/瘢痕]。
3. 心腔[不大/扩大]，室壁[不厚/增厚]。
```

### 脑代谢显像（FDG）

**检查所见要点：**
1. 大脑皮层FDG摄取分布
2. 各脑叶代谢对称性
3. 基底节、丘脑、小脑代谢

**诊断意见模板：**
```
1. [部位]脑皮层FDG代谢[减低/增高]，以[具体脑区]为著。
2. [其他发现]。
3. 结合临床，符合[疾病]代谢模式。
```

### 淀粉样蛋白显像（AV45）

**判读标准：**
- 阳性：灰质摄取≥白质，或特定脑区（额叶、顶叶、颞叶、后扣带回）明显摄取
- 阴性：灰质摄取<白质，以白质摄取为主

**诊断意见模板：**
```
1. 脑内AV45分布[阳性/阴性]。
2. [具体描述摄取模式]。
3. 结合临床，[支持/不支持]脑内β-淀粉样蛋白沉积。
```

### Tau蛋白显像（AV1451）

**判读标准：**
- 主要观察颞叶、海马、后扣带回等区域的摄取
- 需结合AV45结果综合判断

## 四、常见伪影识别

### 生理性摄取
| 部位 | 特点 | 处理 |
|------|------|------|
| 脑 | 正常高摄取 | 无需处理 |
| 心肌 | 可变摄取，空腹可降低 | 禁食、脂餐抑制 |
| 泌尿系统 | 肾脏排泄致肾盂、输尿管、膀胱高摄取 | 利尿、导尿、延迟排尿 |
| 咽部/声带 | 对称性摄取 | 避免说话、咳嗽 |
| 棕色脂肪 | 颈部、锁骨上、纵隔对称摄取 | 保暖、药物抑制 |
| 肌肉 | 运动后或紧张致摄取 | 静卧休息 |

### 病理性伪影
| 类型 | 原因 | 表现 |
|------|------|------|
| 金属伪影 | 假牙、起搏器、假体 | CT图像条状伪影，影响衰减校正 |
| 造影剂伪影 | 增强CT对比剂 | 高密度伪影 |
| 运动伪影 | 呼吸、肠蠕动 | 图像模糊 |
| 注射部位渗漏 | 药物外渗 | 注射点高摄取 |

## 五、报告书写注意事项

1. **客观描述：** 先描述所见，后下结论
2. **定量数据：** 提供SUVmax等客观指标
3. **对比分析：** 与既往检查对比，说明变化
4. **临床结合：** 结合病史、其他检查综合判断
5. **建议明确：** 给出明确的随访或进一步检查建议
6. **术语规范：** 使用标准医学术语，避免口语化
